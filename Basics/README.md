
# 01 - Python Basics

Welcome to the **Python Basics** section!

This folder is the starting point of my Python learning journey. It covers the core concepts every beginner should understand before moving on to advanced topics. Each program is simple, well-commented, and designed to help you learn one concept at a time.


## Programs Included

| File | What You'll Learn |
|------|-------------------|
| `hello_world.py` | Print your first message using the `print()` function. |
| `comments.py` | Learn how to write single-line and multi-line comments. |
| `keywords.py` | Explore Python's reserved keywords using the `keyword` module. |
| `identifiers.py` | Understand the rules for naming variables and identifiers. |
| `indentation.py` | See how indentation controls the flow of a Python program. |
| `first_program.py` | Create a simple program using variables and output statements. |


## Concepts Covered

- Introduction to Python
- Your first Python program
- Using the `print()` function
- Single-line and multi-line comments
- Python keywords
- Identifiers and variable naming rules
- Indentation in Python
- Variables
- Basic output formatting


## Running the Programs

1. Open your Terminal or Command Prompt.
2. Navigate to this folder.

```bash
cd 01_Basics
```

3. Run any Python file.

```bash
python hello_world.py
```

or

```bash
python3 hello_world.py
```

## What You'll Learn

After completing these programs, you'll be able to:

- Write and run your first Python program.
- Display output using the `print()` function.
- Use comments to make your code easier to understand.
- Recognize Python keywords.
- Follow proper variable naming conventions.
- Understand why indentation is important.
- Build a strong foundation for learning Python.
